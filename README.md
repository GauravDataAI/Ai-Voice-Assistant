# 🎙️ AI Powered Voice-Based Assistant

This project was originally developed as my **Major Project during Polytechnic** and is now uploaded on GitHub for **portfolio and documentation purposes**.

## 📌 Overview
This is a voice-based assistant built using **HTML, CSS, and JavaScript**. It can take voice commands, respond through speech, and provide an interactive browser-based experience.

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Web Speech API

## ✨ Features
- Voice recognition
- Speech response
- Interactive user interface
- Browser-based working
- AI-assisted JavaScript logic
